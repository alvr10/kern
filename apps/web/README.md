# KERN - Frontend Marketing Playground

Frontend de la plataforma KERN, un ecosistema diseñado para equipos de marketing para rastrear, gestionar y automatizar contenido multimedia.

## 🚀 Visión General

KERN es un centro operativo que permite a las empresas organizar su flujo de trabajo a través de **Organizaciones** y **Proyectos**. El sistema facilita la colaboración entre miembros para la planificación y ejecución de estrategias de contenido.

## ✨ Características Principales

- **Estructura de Trabajo:** Gestión basada en organizaciones y proyectos con sistema de membresía.
- **Visualización Avanzada:**
  - **Vista Kanban:** Flujo de trabajo ágil para el seguimiento de estados.
  - **Vista de Calendario:** Planificación temporal y visual de publicaciones.
- **Automatización Social:** Integración vía API con cuentas de redes sociales para la programación y subida automática de posts.
- **Modelo Freemium:** Sistema de suscripción para acceso extendido a herramientas de Inteligencia Artificial.
- **Landing Page Premium:** Diseño minimalista y moderno con animaciones de alto rendimiento mediante **GSAP**.

## 🛠️ Stack Tecnológico

- **Core:** [Next.js 15+](https://nextjs.org/)
- **Lenguaje:** TypeScript (Strict Mode)
- **Animaciones:** [GSAP](https://greensock.com/gsap/) & [@gsap/react](https://github.com/greensock/react)
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)

## 🏁 Inicio Rápido

1. Instalación de dependencias:

```bash
pnpm install
```

2. Ejecución en entorno de desarrollo:

```bash
pnpm dev
```

3. Construcción para producción:

```bash
pnpm build
```
